# Archive

This directory contains experimental and aspirational code from the original
ASI:BUILD research project (v1, August 2025). These modules were moved here
during the v2.0 restructure because they are:

- **Template-generated scaffolding** (e.g., `ultimate_emergence/` — 60 files
  from an identical template)
- **Duplicate directories** (e.g., `consciousness_duplicate/` — byte-for-byte
  copy of `consciousness_engine/`)
- **Aspirational modules** without working implementations (e.g., `telepathy/`,
  `omniscience/`, `multiverse/`)
- **Empty stubs** (directories containing only `.gitkeep`)

The real, tested code lives in `src/asi_build/`. If you find something useful
here, consider extracting it into a proper module with tests.
