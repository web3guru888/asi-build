# VLA++ Data Directory

The VLA++ training data (5.9GB) has been excluded from version control.

## Required Data:
- COCO dataset annotations
- Model checkpoints
- Training datasets

## Setup Instructions:
1. Download COCO dataset to data/coco/
2. Place model checkpoints in checkpoints/
3. Configure paths in config files

Note: Large binary files should be stored in a data repository or cloud storage.
